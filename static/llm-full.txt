## Architecture

### Frontend Architecture
- **Router**: SvelteKit file-based routing
- **State Management**: Svelte 5 runes ($state, $props, $derived)
- **Data Loading**: Server-side with +page.server.ts files
- **Components**: Reusable components (SEOHead, Navbar, Footer)
- **Styling**: Utility-first with TailwindCSS, scoped styles where needed

### Content Management
- **Ghost CMS Integration**: 
  - Headless CMS for blog posts and projects
  - Content API v6.10
  - Tag-based content organization (projects, notes)
  - Featured content selection
  - Reading time calculation
  - Excerpt generation

### Deployment
- **Platform**: Cloudflare Pages
- **Build**: Static site generation with server-side rendering
- **Edge Runtime**: Cloudflare Workers for server functions
- **Performance**: Global CDN, edge caching
- **Configuration**: wrangler.toml for deployment settings
