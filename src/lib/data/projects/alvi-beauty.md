<p><strong>Alvi Beauty</strong> is a multi-platform beauty service booking product I worked on as a contracted developer for Romarianec Tech. It lets users book beauty services from home while giving service providers and admins the tools they need to manage everything smoothly.</p><p>The platform includes four mobile apps (iOS and Android for both users and providers), a web booking experience, a provider dashboard, and a super admin dashboard. My role covered both backend development and key parts of the overall system architecture.</p><p>The main challenge was building something that felt consistent across mobile apps, web, and dashboards, without it becoming messy or hard to scale. Everything needed to work together cleanly.</p><p>I built the backend with Node.js and MongoDB to handle users, bookings, and provider data reliably at scale. The mobile apps were built with React Native, making it easier to support both iOS and Android from a single codebase. Service providers manage schedules and bookings through the web dashboard, while admins have full visibility and control through the super admin panel.</p><p>Overall, Alvi Beauty simplified how people discover and book beauty services, while giving providers better tools to run their businesses. It’s a solid example of the kind of end-to-end, cross-platform systems I enjoy building.</p><p><a href="https://alvibeauty.com/en-ua" rel="noreferrer">Live Project</a></p>
